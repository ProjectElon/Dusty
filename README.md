# Dusty

### A Simple 3D Software Renderer

### What I Learned :
 - Line Drawing.
 - Triangle Rasterization.
 - Texture Mapping (with Perspective correction).
 - Loading OBJ files.
 - Perspective Projection.
 - Backface Culling.
 - Clipping.
 - Depth Testing (Z Buffer).
 - Dynamic Lighting (Phong Lighting Model)
